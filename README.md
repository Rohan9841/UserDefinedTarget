# UserDefinedTarget

This is a unty project for android app that allows user to place a certain 3d object in some user defined target. 

#Getting Started in Unity:
1. Clone the repo 
2. Open ARUserDef.unity in unity: AR7/Assets/DataFiles/Scenes/ARUserDef.unity

# Getting started in android:
1. Download AR_Astronaut.apk from the repo and install in your android phone 
2. Use the app and tap on camera button on bottom left while pointing the camera toward any image. An astronaut will appear from the image. 

#Built with:

Unity, Visual Studio, Vuforia, c#
